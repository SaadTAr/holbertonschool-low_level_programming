# Doubly Linked Lists

This project is about doubly linked lists in C.

## Description
A doubly linked list is a data structure where each node contains:
- An integer value
- A pointer to the previous node
- A pointer to the next node

## Tasks
- Print list
- List length
- Add node
- Add node at the end
- Free list
- Get node at index
- Sum list
- Insert at index
